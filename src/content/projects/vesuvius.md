## Overview

Vesuvius is a long-term game project where my work centers on technical audio implementation and gameplay-facing audio systems.

## Technical Focus

- Wwise events, states, switches, and parameters
- State-driven ambience and room-aware transitions
- Combat, exploration, UI, and interaction feedback
- Runtime event pipelines designed for iteration

## My Role

I lead audio software development for the project, connecting gameplay state to audio behavior and building implementation patterns that can be extended as the game grows.
